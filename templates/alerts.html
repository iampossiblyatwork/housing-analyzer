How alerts work

Alerts are evaluated each time you run a Market Analysis for the matching ZIP code. If the condition is met, an SMS is sent via Twilio.

To enable SMS: add TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_FROM_NUMBER, and ALERT_TO_NUMBER to your .env file.
